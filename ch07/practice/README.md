## p02.c
编写一个程序，确认只要没有发生高位溢出，则：  
无符号整数位左移后的值等于其乘以2的指数幂后的值。  
无符号整数位右移后的值等于其除以2的指数幂后的值。

## p03.c

## p04.c

编写reset函数，返回将无符号整数x的第pos位设为0后的值。  
编写inverse函数，返回将无符号整数x的第pos位取反后的值。

## p05.c

编写inverse_n函数，返回将无符号整数x的第pos位至第pos+n-1位的n位取反后的值。
